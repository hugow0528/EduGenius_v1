## 🎓 關於平台

**EduGenius AI 智學**是一個先進的多科目AI學習平台，專為香港中學文憑試(DSE)學生而設計。平台運用最先進的人工智能技術，為六大主要科目提供個人化學習輔助，協助學生在學業上取得卓越成就。

我們的使命是透過智能輔導系統，讓優質教育變得更容易獲取、更具吸引力、更有效率，並且能夠適應每位學生的學習風格和需求。

### 核心亮點

- **🤖 AI驅動學習**：採用先進AI模型進行智能輔導
- **📚 多科目支援**：全面涵蓋DSE考試科目
- **🌐 雙語介面**：支援英文及繁體中文
- **📱 響應式設計**：無縫支援桌面、平板及手機裝置
- **🔒 安全認證**：基於Firebase的用戶認證系統
- **🌙 深色模式**：護眼深色主題，舒適學習
- **♿ 無障礙**：遵循WCAG無障礙標準設計

## ✨ 功能特色

### 🎯 個人化學習體驗

- **自適應AI回應**：根據您的科目和學習程度提供量身定制的答案
- **互動式對話介面**：自然對話風格的學習方式
- **圖像分析**：上傳或貼上圖片進行AI分析和問題解決
- **OCR支援**：使用Tesseract.js自動提取圖片中的文字
- **Markdown格式**：清晰、結構化的回應及適當格式

### 🛠️ 智能學習工具

- **問AI**：獲得針對特定科目問題的全面答案
- **字典AI**：詳細解釋術語、概念和詞彙
- **導學習**：循序漸進的指導，不直接提供答案
- **專科工具**：每個科目的專門工具（作文評分器、試卷生成器、程式碼審查等）

### 💡 用戶友善介面

- **直觀導航**：易用的科目選擇和工具導航
- **即時回應**：快速的AI驅動答案
- **複製與分享**：輕鬆複製或分享AI回應
- **訊息歷史**：保持對話脈絡以提供更好的協助
- **鍵盤快捷鍵**：高效的鍵盘導航

### 🔐 安全與隱私

- **Firebase認證**：安全的用戶註冊和登入
- **受保護內容**：僅限已認證用戶訪問
- **安全API通訊**：與AI服務的加密通訊

## 🤖 AI工具

### 1. **問AI** 🤖
獲得針對特定科目問題的全面、詳細答案。AI提供清晰的解釋、實用範例和相關背景資訊。

**使用情境：**
- 理解複雜概念
- 獲得詳細解釋
- 分析圖片和圖表
- 循序漸進地解決問題

### 2. **字典AI** 📖
查找術語、概念和詞彙的詳細解釋。提供定義、使用範例和背景資訊。

**使用情境：**
- 學習新術語
- 理解技術名詞
- 探索概念定義
- 獲得雙語解釋

### 3. **導學習** 🧭
獲得引導性提示和策略方向，而不直接提供答案，鼓勵批判性思考和獨立學習。

**使用情境：**
- 發展解決問題的能力
- 通過引導學習
- 建立信心
- 理解思考過程

### 4. **專科工具**

每個科目都有額外的專門工具：

- **中文**：文樞古典文學平台
- **英文**：寫作練習、詞彙生成器、詞彙測驗
- **數學/物理/生物**：試卷生成器
- **ICT**：程式碼審查平台

## 🚀 快速開始

### 系統要求

- 現代網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- 有效的網絡連接
- 電子郵件地址（用於註冊）

### 訪問平台

1. **造訪平台**：導航至EduGenius平台網址
2. **建立帳戶**：使用電子郵件和密碼註冊
3. **登入**：使用您的憑證登入
4. **選擇科目**：從主畫面選擇您的科目
5. **選擇工具**：選擇您需要的AI工具
6. **開始學習**：開始提問並與AI互動

### 使用AI工具

#### 問AI
```
1. 在輸入欄位中輸入您的問題
2. （選用）上傳或貼上圖片
3. 按「傳送」或按Enter鍵
4. 查看AI回應
5. 根據需要繼續對話
```

#### 字典AI
```
1. 輸入術語或概念
2. 按「定義」
3. 閱讀詳細解釋
4. 探索相關範例
```

#### 導學習
```
1. 提出您的問題
2. 接收引導提示
3. 思考問題
4. 提出後續問題
5. 在引導下得出解答
```

### 鍵盤快捷鍵

- **Enter**：傳送訊息
- **Shift + Enter**：在訊息中換行
- **Escape**：返回上一個畫面
- **Ctrl/Cmd + V**：從剪貼簿貼上圖片

## 🏗️ 平台架構

### 前端
- **HTML5**：語義化標記，確保無障礙
- **TailwindCSS**：實用優先的CSS框架
- **原生JavaScript**：現代ES6+ JavaScript
- **Font Awesome**：圖示庫
- **Google Fonts**：Inter、Noto Serif TC、Noto Sans TC

### 後端服務
- **Firebase認證**：用戶管理和認證
- **Pollinations AI API**：使用Gemini模型的AI驅動回應
- **Tesseract.js**：客戶端OCR圖片文字提取

### 關鍵功能
- **漸進式網頁應用程式(PWA)**：可安裝在行動裝置上
- **響應式設計**：行動優先方法
- **深色模式支援**：系統和手動主題切換
- **無障礙**：ARIA標籤、鍵盤導航、螢幕閱讀器支援
