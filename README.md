## üöÄ Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Active internet connection
- Email address for registration

### Access the Platform

1. **Visit the Platform**: Navigate to the EduGenius platform URL
2. **Create Account**: Register with your email and password
3. **Sign In**: Log in with your credentials
4. **Choose Subject**: Select your subject from the main screen
5. **Select Tool**: Choose the AI tool you need
6. **Start Learning**: Begin asking questions and interacting with AI

### Using AI Tools

#### Ask AI
```
1. Type your question in the input field
2. (Optional) Upload or paste an image
3. Click "Send" or press Enter
4. Review the AI response
5. Continue the conversation as needed
```

#### Dictionary AI
```
1. Enter a term or concept
2. Click "Define"
3. Read the detailed explanation
4. Explore related examples
```

#### Guide Learning
```
1. Ask your question
2. Receive guiding hints
3. Think through the problem
4. Ask follow-up questions
5. Arrive at the solution with guidance
```

### Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: New line in message
- **Escape**: Return to previous screen
- **Ctrl/Cmd + V**: Paste image from clipboard
